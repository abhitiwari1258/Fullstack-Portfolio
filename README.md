# 🚀 Full Stack Portfolio Website

## 📌 Project Overview
This is a full-stack portfolio website built to showcase my projects, skills, and experience. It includes a dynamic project section, contact form, and an admin panel to manage content.

---

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router DOM
- Axios

### Backend (Planned)
- Node.js
- Express.js

### Database (Planned)
- MongoDB

---

## ✨ Features

### 👤 User Side
- View portfolio projects
- About page
- Contact form

### 🔐 Admin Side (Upcoming)
- Admin login
- Add/Edit/Delete projects
- View contact messages
